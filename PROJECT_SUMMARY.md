## 🎨 Color Customization

Default color scheme: **Sky Blue**
- Primary Color: `#0ea5e9`
- Primary Shade: `#0284c7`

To change colors, edit `tailwind.config.js`:

```javascript
colors: {
  primary: {
    500: '#your-color',  // Change this
    600: '#darker-shade'
  }
}
```

Popular alternatives:
- 💜 Purple: `#a855f7`
- 💚 Emerald: `#10b981`
- ❤️ Rose: `#f43f5e`
- 💙 Indigo: `#6366f1`
